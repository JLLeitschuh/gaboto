While {@link org.oucs.gaboto.entities.GabotoEntity}s are representation of Gaboto objects at a specific point in time, {@link org.oucs.gaboto.entities.time.GabotoTimeBasedEntity} represent an Gaboto object over its lifespan.

{@link org.oucs.gaboto.entities.time.GabotoTimeBasedEntity}s are used to set and change timebound properties or can be used to display the changes of an entity over time.
